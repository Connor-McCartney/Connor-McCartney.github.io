# 1. Solutions of first order ODEs


Directly integrable:

$\frac{dy}{dx} = f(x)$


Seperable:

$\frac{dy}{dx} = f(x) g(y)$

Linear:

$\frac{dy}{dx} = q(x) - p(x) y$


<br> <br> <br>

Exercises:

1:

$\frac{dy}{dx} = y^{\frac{2}{3}}, \ y(0) = 1$

$y = \int{y^{\frac{2}{3}}} \ dx$


Final answer:

$y = \frac{1}{27} (x+3)^3$



2:

$\frac{dy}{dx} = \frac{3x^2 + 4x + 2}{2(y-1)}, \ y(0) = 1$


Final answer: 

$y = 1 \pm sqrt{x^3 + 2x^2 + 2x}, \ \ x > 0$
